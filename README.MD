## NodeJS Test Phones API

### Config
1. Create config 
    ```bash
    cp src/config/constants.json.sample src/config/constants.json
    ```
2. Fill it with correct values

### Start application
```yarn start```

### Run tests
```yarn test```

### Linter
```yarn lint```
